About WES TEK

Founded in 2025 by Canadian developer Wesley Synnott, WES TEK is a lean, single-developer studio dedicated to building native iOS apps for Apple devices. Projects are organized around a modern MVVM architecture with an emphasis on clean structure, performance, and user-focused design. The studio's upcoming app, The Wastelander, brings a companion-style toolkit to Fallout 76 players on iOS—combining checklist-driven progress tracking, interactive maps, guides, and more into a single, intuitive hub.

Wesley Synnott

Developer

Wesley Synnott is a Canadian iOS developer who specializes in Swift-first, SwiftUI-driven applications built on clean MVVM principles. His work emphasizes modular architecture, maintainable code, and seamless data integration using technologies like Core Data, iCloud, CloudKit, and Firebase.

He brings a disciplined, user-centred approach to development—combining technical rigour with intuitive interface design to create scalable iOS applications built to perform.
